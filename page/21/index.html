在 macOS 下怎么样安装、快速切换多个 jdk 版本

发表 2020-01-17 晚上 分类 Java

说明：

  1. jdk 在 macOS 下的安装路径为 /Library/Java/JavaVirtualMachines/；

  2. /usr/libexec/java_home 是 macOS 下提供的工具，类似一个简单的 java 版本管理工具，可使用 -h 参数查看使用帮助。
