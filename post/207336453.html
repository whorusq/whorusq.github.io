五、关键理解（避免常见误区）

  1. Agent ≠ 模型
  • LLM 只是“大脑”
  • Agent 是“操作系统”
  1. RAG ≠ 数据库查询
  • 本质是：语义检索 + prompt 拼接
  1. Skill vs Tool
  • Tool：原子能力（API）
  • Skill：组合能力（可能包含多个 Tool）
  1. Prompt 是“代码”
  • 在很多系统里，Prompt ≈ DSL（领域语言）
